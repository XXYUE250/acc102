## 📌 Project Overview
This interactive data application is designed for Regional Sales Managers and Business Analysts to explore historical sales data, identify commercial trends, and make data-driven decisions. 

It was developed as the **Track 4 (Interactive Data Tool)** submission for the **ACC102 Mini Assignment** at Xi'an Jiaotong-Liverpool University (XJTLU).

## ✨ Key Features
* **Interactive Filtering:** Dynamic sidebar to slice data by Year, Region, and Product Category.
* **Automated Smart Insights:** Python-generated text summarizing the top-performing State, Month, and Category based on real-time filters.
* **Comprehensive Visualizations:** Includes time-series sales trends, customer segmentation (Donut chart), and sub-category performance (Bar charts).
* **Geographical & Hierarchical Analysis:** Features an interactive choropleth map of the USA and a Treemap breaking down category hierarchies.
* **Raw Data Explorer & Export:** Users can view the underlying dataset and export their customized filtered data as a CSV file with one click.
