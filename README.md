## 📌 Project Overview
This interactive data application is designed for **Regional Sales Managers** and **Business Analysts** to explore historical sales data, identify commercial trends, and make data-driven decisions. The tool transforms raw transactional records into actionable intelligence.

It was developed as the **Track 4 (Interactive Data Tool)** submission for the **ACC102 Mini Assignment** (2025-26 S2) at Xi'an Jiaotong-Liverpool University (XJTLU).

## ✨ Key Features
* **Interactive Data Slicing:** A dynamic sidebar allowing users to filter the entire dashboard by Year, Region, and Product Category simultaneously.
* **🤖 Automated Smart Insights:** Utilizes Python logic to auto-generate summary text, identifying the top-performing State, Month, and Category in real-time based on selected filters.
* **📊 Strategic KPI Metrics:** Instant visibility into Total Sales, Order Volume, Customer Count, and Average Transaction Value.
* **📈 Multi-Tab Analysis:**
    * **Business Overview:** Visualizes sales trends over time, customer segments, and sub-category performance.
    * **Geography & Hierarchy:** Features an interactive USA choropleth map and a Treemap representing the hierarchy between product categories and sub-categories.
* **📁 Data Governance & Export:** Allows users to inspect the filtered dataset and export it as a CSV file for offline use in Microsoft Excel.

---

## 🛠️ Tech Stack
* **Python**: Primary language for data logic and backend operations.
* **Streamlit**: Web framework for building the interactive user interface.
* **Pandas**: Used for substantive data cleaning (handling encoding and date formatting), transformation, and aggregation.
* **Plotly Express**: Employed for rendering high-quality, interactive geographical maps and hierarchical visualizations.
